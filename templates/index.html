Welcome to the CIFAR Classifier

This web app allows you to upload an image, select a model, and classify it into one of the CIFAR-10 categories.

The two models are a vision transformer model that uses patchsize=4 and the traditional attention kernel,

and a Performer-Relu model that uses fast attention mechanisms.

By approximating the regular attention mechanism via random feature maps, Performers can estimate regular (softmax) full-rank-attention Transformers with provable accuracy, but using only linear (as opposed to quadratic) space and time complexity.
